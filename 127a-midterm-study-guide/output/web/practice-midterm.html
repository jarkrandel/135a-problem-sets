    Problems 2 Practice midterm

    Here’s a couple of practice problems to give you an idea of what to expect.
    🔗

    1.

    Consider the sequence \((a_n)\) given by
    🔗
    \begin{equation*} a_n = \frac{2n^2+1}{n^2+3}. \end{equation*}
    Prove that \((a_n)\) converges to \(2\) directly using the \(\epsilon\)-\(N\) definition of the limit.
    🔗
    Solution.
    Notice that
    🔗
    \begin{equation*} \begin{aligned} |a_n - 2| &= \left|\frac{2n^2+1}{n^2+3} - 2\right| = \left|\frac{2n^2+1-2(n^2+3)}{n^2+3}\right|\\ &= \left|\frac{-5}{n^2+3}\right| = \frac{5}{n^2+3} < \frac{5}{n^2}. \end{aligned} \end{equation*}
    Therefore, for any \(\epsilon > 0\text{,}\) if we know that \(n\) is large enough so that \(\frac{5}{n^2} < \epsilon\text{,}\) then we will have \(|a_n-2| < \epsilon\text{.}\) By rearranging to solve for \(n\text{,}\) \(\frac{5}{n^2} < \epsilon\) if and only if \(\sqrt{5/\epsilon} < n\text{.}\) So, we should choose \(N > \sqrt{5/\epsilon}\) in our proof.
    🔗
    Proof: Let \(\epsilon > 0\text{.}\) Let \(N \in \mathbb{N}\) satisfy \(N > \sqrt{5/\epsilon}\) (such an integer exists by the Archimedean Property). Then for any \(n \geq N\text{,}\) the above work shows that
    🔗
    \begin{equation*} \begin{aligned} |a_n-2| < \frac{5}{n^2} \leq \frac{5}{N^2} < \epsilon. \end{aligned} \end{equation*}
    🔗
    🔗

    2.

    Let \(A,B \subseteq (0,\infty)\) be nonempty and bounded above. Define
    🔗
    \begin{equation*} AB = \{ab : a \in A,\ b \in B\}. \end{equation*}
    Prove that \(\sup(AB) = \sup A \cdot \sup B\text{.}\) You may assume that \(\sup A > 0\) and \(\sup B > 0\text{.}\)
    🔗
    Solution.
    Let \(s = \sup A\) and \(t = \sup B\) to simplify notation. We need to show that \(st\) is the least upper bound for \(AB\text{.}\) First, we show it is an upper bound. For any \(a \in A\) and \(b \in B\text{,}\) we know that \(a \leq s\) and \(b \leq t\) since \(s\) is an upper bound for \(A\) and \(t\) is an upper bound for \(B\text{.}\) Therefore, for any \(a \in A\) and \(b \in B\text{,}\)
    🔗
    \begin{equation*} ab \leq sb \leq st. \end{equation*}
    This shows that \(st\) is an upper bound. Now, we show it is the least upper bound. We can do this directly. Let \(u\) be an upper bound for \(AB\text{.}\) Then for any \(a \in A\) and \(b \in B\)
    🔗
    \begin{equation*} ab \leq u. \end{equation*}
    This gives
    🔗
    \begin{equation*} a \leq \frac{u}{b} \end{equation*}
    for any fixed \(b \in B\) and for all \(a \in A\text{.}\) This shows that \(u/b\) is an upper bound for \(A\text{,}\) hence \(s \leq u/b\) since \(s\) is the least upper bound. Rearranging this, we get
    🔗
    \begin{equation*} b \leq \frac{u}{s}. \end{equation*}
    for any \(b \in B\text{,}\) meaning that \(u/s\) is an upper bound for \(B\text{.}\) Since \(t\) is the least upper bound for \(B\text{,}\) we get \(t \leq u/s\text{,}\) hence \(st \leq u\text{.}\) This shows \(st\) is the least upper bound for \(AB\text{.}\)
    🔗
    🔗

    3.

    Determine whether the following statements are True or False.
    🔗
    1. There exists a Cauchy sequence with an unbounded subsequence.
      🔗
      🔗
    2. If \((x_n)\) and \((y_n)\) are convergent sequences such that \(x_n \to x\text{,}\) \(y_n \to y\text{,}\) and \(x_n < y_n\) for all \(n \in \mathbb{N}\text{,}\) then \(x < y\text{.}\)
      🔗
      🔗
    3. The set of all rational numbers \(p/q\) with \(q=2^n\) for some \(n \in \mathbb{N}\) is dense in \(\mathbb{R}\text{.}\)
      🔗
      🔗
    4. If \((a_n)\) has a subsequence that does not converge, then \((a_n)\) does not converge.
      🔗
      🔗
    5. The series \(-1 + \frac{1}{1^2} - \frac{1}{2} + \frac{1}{2^2} - \frac{1}{3} + \frac{1}{3^2} - \ldots\) converges.
      🔗
      🔗
    Solution.
    1. False. Every Cauchy sequence converges by the Cauchy Criterion, and hence is bounded. Every subsequence of a bounded sequence is bounded.
      🔗
      🔗
    2. False. The statement is true by the Order Limit Theorem if \(x < y\) is replaced with \(x \leq y\text{,}\) but for this statement we can use \(x_n = 0\) and \(y_n = \frac{1}{n}\) as a counterexample since \(x = y = 0\text{.}\)
      🔗
      🔗
    3. True. We have no restriction on the numerators, and the denominators get arbitrarily large. This is enough for density by the proof of density.
      🔗
      🔗
    4. True. The contrapositive of this statement is: If \((a_n)\) converges, then every subsequence converges. We proved that every subsequence of a convergent sequence converges to the same limit, so this is true.
      🔗
      🔗
    5. False. The series consisting of odd terms is \(\sum(-1/n)\text{,}\) while the even terms is \(\sum 1/n^2\text{.}\) The sum of even terms converges while the sum of odd terms diverges to \(-\infty\text{,}\) implying the whole series diverges to \(-\infty\text{.}\)
      🔗
      🔗
    🔗
    🔗

    4.

    Let \((x_n)\) be an increasing sequence such that \(x_n \geq 0\) for all \(n \in \mathbb{N}\) and there exists a subsequence \((x_{n_k})\) of \((x_n)\) that is bounded.
    🔗
    1. Prove that \((x_n)\) is bounded.
      🔗
      🔗
    2. Use the result of part (a) to prove that \((x_n)\) converges.
      🔗
      🔗
    3. Let \(x \in \mathbb{R}\) be such that \(x_n \to x\text{.}\) Prove that \(x_n \leq x\) for all \(n \in \mathbb{N}\text{.}\) Hint: proof by contradiction is one way of doing this.
      🔗
      🔗
    Solution.
    1. Since \((x_{n_k})\) is bounded, there exists \(M > 0\) such that \(|x_{n_k}| \leq M\) for all \(k \geq 1\text{.}\) Since \(x_n \geq 0\) for all \(n\text{,}\) we know that \(|x_n| = x_n\) for all \(n\text{.}\) Given any fixed \(n\text{,}\) we can find \(n_k > n\) so that
      🔗
      \begin{equation*} |x_n| = x_n \leq x_{n_k} = |x_{n_k}| \leq M \end{equation*}
      using the fact that \((x_n)\) is increasing.
      🔗
      🔗
    2. Since \((x_n)\) is increasing and bounded, it converges by the Monotone Convergence Theorem.
      🔗
      🔗
    3. Towards a proof by contradiction, suppose that \(x_m > x\) for some \(m \in \mathbb{N}\text{.}\) Then \(\epsilon = x_m - x > 0\text{,}\) and the fact that \((x_n) \to x\) means that there exists \(N \in \mathbb{N}\) such that \(|x_n-x| < \epsilon\) for all \(n \geq N\text{.}\) On the other hand, the fact that \((x_n)\) is increasing means that \(x_n \geq x_m > x\) for all \(n \geq m\text{,}\) meaning \(|x_n - x| > |x_m-x| = \epsilon\) for all \(n \geq m\text{.}\) So, for any \(n \geq \max\{N,m\}\text{,}\) we get
      🔗
      \begin{equation*} \epsilon < |x_n-x| < \epsilon, \end{equation*}
      a contradiction.
      🔗
      🔗
