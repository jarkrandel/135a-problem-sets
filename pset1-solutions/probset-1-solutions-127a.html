    Problem 1 (Abbott Example 1.2.5 and Exercise 1.2.6): Look up Example 1.2.5 in the textbook, and then complete Exercise 1.2.6. This is very important for everything, so please work to understand it.
    🔗
    1. Verify the triangle inequality in the special case where \(a\) and \(b\) have the same sign.
      🔗
      🔗
    2. Find an efficient proof for all cases at once by first demonstrating \((a+b)^2 \le (|a| + |b|)^2\text{.}\) For part (b), you can use the fact that \(f(x)=\sqrt{x}\) is increasing.
      🔗
      🔗
    3. Prove \(|a-b| \le |a-c| + |c-d| + |d-b|\) for all \(a\text{,}\) \(b\text{,}\) \(c\text{,}\) and \(d\text{.}\)
      🔗
      🔗
    4. Prove \(||a|-|b|| \le |a-b|\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. If \(a\) and \(b\) have the same sign, then they are either both positive or both negative. If both are positive, then \(a+b\) is also positive, so
      🔗
      \begin{equation*} |a + b| = a + b = |a| + |b|. \end{equation*}
      In particular, \(|a+b|\leq |a| + |b|\text{.}\) If they are both negative, then \(a+b\) is negative, so
      🔗
      \begin{equation*} |a+b| = -(a+b) = (-a) + (-b) = |a| + |b|. \end{equation*}
      🔗
    2. Notice that
      🔗
      \begin{equation*} (a+b)^2 = a^2 + 2ab + b^2 \leq a^2 + 2|a||b| + b^2 = |a|^2 + 2|a||b| + |b|^2 = (|a| + |b|)^2. \end{equation*}
      Since \(\sqrt{x}\) is an increasing function and \((a+b)^2 = |a+b|^2\text{,}\) we get \(|a+b| \leq |a| + |b|\text{.}\)
      🔗
      🔗
    3. We use the delightful trick of adding zero to get
      🔗
      \begin{equation*} \begin{aligned}|a-b| & = |a + 0 + 0 -b|\\ & = |a + (-c+c) + (-d+d) -b|\\ & = |(a-c) + (c-d + d-b)|.\end{aligned} \end{equation*}
      Applying the triangle inequality twice, we get
      🔗
      \begin{equation*} \begin{aligned}|a-b| \leq |a-c| + |(c-d)+(d-b)| \leq |a-c| + |c-d| + |d-b|.\end{aligned} \end{equation*}
      🔗
    4. Recall that, by the definition of the absolute value,
      🔗
      \begin{equation*} ||a|-|b|| = \begin{cases} |a|-|b| & \text{if $|a|-|b| {\gt} 0$}\\ -(|a|-|b|) & \text{if $|a|-|b| \leq 0$} \end{cases} \end{equation*}
      So it suffices to prove both \(|a|-|b| \leq |a-b|\) and \(|b|-|a|=-(|a|-|b|)\leq |a-b|\text{.}\)
      🔗
      Let’s begin with the first inequality, and let’s use Abbott’s suggestion. The triangle inequality gives
      🔗
      \begin{equation*} |a| = |a-b+b| \leq |a-b| + |b|. \end{equation*}
      Rearranging this gives \(|a|-|b| \leq |a-b|\text{.}\) The second inequality follows by using the same strategy with \(a\) and \(b\) flipped. Indeed, the triangle inequality gives
      🔗
      \begin{equation*} |b| = |b-a+a| \leq |b-a| + |a| = |a-b| + |a| \end{equation*}
      so that rearranging gives
      🔗
      \begin{equation*} |b|-|a| \leq |a-b|. \end{equation*}
      🔗
    Problem 2 (Abbott Exercise 1.3.6): Given sets \(A\) and \(B\text{,}\) define \(A+B = \{a+b : a \in A \text{ and } b \in B\}\text{.}\) Follow these steps to prove that if \(A\) and \(B\) are nonempty and bounded above then \(\sup(A + B) = \sup A + \sup B\text{.}\)
    🔗
    1. Let \(s = \sup A\) and \(t = \sup B\text{.}\) Show \(s+t\) is an upper bound for \(A+B\text{.}\)
      🔗
      🔗
    2. Now let \(u\) be an arbitrary upper bound for \(A+B\text{,}\) and temporarily fix \(a \in A\text{.}\) Show \(t \le u-a\text{.}\)
      🔗
      🔗
    3. Finally, show \(\sup(A+B) = s+t\text{.}\)
      🔗
      🔗
    4. Construct another proof of this same fact using Lemma 1.3.8.
      🔗
      🔗
    Solution:Remember our big picture strategy for proving that the supremum of some set (in this case \(A+B\)) is equal to some number (in this case \(\sup A + \sup B\)): We must show that \(\sup A + \sup B\) is an upper bound for \(A+B\text{,}\) then we must show that it is less than or equal to any other upper bound.
    🔗
    1. To show that \(s+t\) is an upper bound for \(A+B\text{,}\) we must show that it is greater than or equal to any element in \(A+B\text{.}\) So, let \(v\in A+B\text{.}\) By definition, there exist \(a\in A\) and \(b\in B\) such that \(v = a + b\text{.}\) Since \(s\) is an upper bound for \(A\) and \(b\) is an upper bound for \(B\text{,}\) we know that \(a \leq s\) and \(b \leq t\text{.}\) Adding these inequalities gives
      🔗
      \begin{equation*} v = a+b \leq s+t. \end{equation*}
      🔗
    2. \(t = \sup B\) and we want to show \(t \leq u-a\text{.}\) Since \(t\) is the least upper bound for \(B\text{,}\) we know that it is less than or equal to any upper bound, so we would be done if we could show that \(u-a\) is an upper bound for \(B\text{!}\) By the definition of \(u\) as an upper bound for \(A+B\text{,}\) we know that
      🔗
      \begin{equation*} a+b \leq u \text{ for all } b\in B. \end{equation*}
      Rearranging this gives
      🔗
      \begin{equation*} b \leq u-a \text{ for all } b\in B, \end{equation*}
      proving that \(u-a\) is in fact an upper bound for \(B\text{.}\) Since \(t\) is the least upper bound for \(B\text{,}\) we know that
      🔗
      \begin{equation*} t \leq u - a. \end{equation*}
      🔗
    3. We still need to prove the lesser part: For any upper bound \(u\) of \(A+B\text{,}\) \(s+t \leq u\text{.}\) Rearranging the conclusion of the previous part, (and using the fact that it holds for any fixed \(a\in A\)) we see that
      🔗
      \begin{equation*} a \leq u-t \text{ for any } a\in A \end{equation*}
      This proves that \(u-t\) is an upper bound for \(A\text{,}\) and so \(s = \sup A\) satisfies \(s \leq u-t\text{.}\) Therefore, \(s+t \leq u\) as desired.
      🔗
      🔗
    4. Here’s the idea: We can apply Lemma 1.3.8 to \(A\) and \(B\) separately to find elements of \(a\) and \(b\) very very close to \(s\) and \(t\) separately. This means \(a+b\) is very close to \(s+t\text{,}\) which we can use to prove \(s+t\) satisfies the condition in Lemma 1.3.8 for \(A+B\text{.}\)
      🔗
      By Lemma 1.3.8, it suffices to prove the following statement: For every \(\epsilon {\gt} 0\text{,}\) there exists an element \(v\in A+B\) such that \(\sup A + \sup B-\epsilon {\lt} v\text{.}\) Towards a proof of this, fix \(\epsilon {\gt} 0\text{.}\) Applying Lemma 1.3.8 to the sets \(A\) and \(B\) with constant \(\epsilon /2\text{,}\) we see that there exist \(a\in A\) and \(b\in B\) such that \(\sup A - \epsilon /2 {\lt} a\) and \(\sup B - \epsilon /2 {\lt} b\text{.}\) Adding these together, we see that
      🔗
      \begin{equation*} a+b {\gt} (\sup A - \epsilon /2) + (\sup B -\epsilon /2) = \sup A + \sup B -\epsilon \end{equation*}
      as desired.
      🔗
      🔗
    Problem 3 (Abbott Exercise 1.3.9):
    🔗
    1. If \(\sup A < \sup B\text{,}\) show that there exists an element \(b \in B\) that is an upper bound for \(A\text{.}\)
      🔗
      🔗
    2. Give an example to show that this is not always the case if we only assume \(\sup A \le \sup B\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. Since \(\sup A {\lt} \sup B\text{,}\) we know that \(\epsilon = \sup B - \sup A {\gt} 0\text{.}\) Therefore, we can use Lemma 1.3.8 to get the existence of \(b\in B\) such that
      🔗
      \begin{equation*} b {\gt} \sup B -\epsilon = \sup B - (\sup B -\sup A) = \sup A. \end{equation*}
      Since \(\sup A\) is an upper bound for \(A\text{,}\) this means that
      🔗
      \begin{equation*} a \leq \sup A {\lt} b \text{ for every } a\in A, \end{equation*}
      which shows that \(b\) is an upper bound for \(A\text{.}\)
      🔗
      🔗
    2. Let \(A = B = (0,1)\text{.}\) Then \(\sup A = \sup B = 1\text{,}\) yet no element of \(B\) is an upper bound for \(A\text{.}\) Indeed, for any \(b\in B\text{,}\) \(b {\lt} 1\) so that the element \(a = \frac{b+1}{2}\in A\) (\(a\) is just the average of \(b\) and \(1\)) satisfies
      🔗
      \begin{equation*} b = \frac{b+b}{2} {\lt} \frac{b+1}{2} = a. \end{equation*}
      🔗
    Problem 4 (Abbott Exercise 1.4.4): Let \(a < b\) be real numbers and consider the set \(T = \Q \cap [a,b]\text{.}\) Show \(\sup T = b\text{.}\) Begin by justifying the fact that \(\sup T\) exists.
    🔗
    Solution:
    🔗
    First of all, since \(T \subset [a,b]\text{,}\) we know that \(t \leq b\) for all \(t\in T\text{.}\) This shows that \(b\) is an upper bound for \(T\text{.}\) On the other hand, the density of \(\mathbb {Q}\) in \(\R \) implies that there exists \(r\in \Q \cap (a,b)\text{,}\) showing that \(T\) is nonempty. Therefore, \(\sup T\) exists by the Axiom of Completeness.
    🔗
    Using Lemma 1.3.8, in order to show that \(b\) is the least upper bound it suffices to show that for any \(\epsilon {\gt} 0\text{,}\) there exists an element \(t\in T\) satisfying \(b - \epsilon {\lt} t\text{.}\) Towards a proof of this last statement, fix some \(\epsilon {\gt} 0\) such that \(\epsilon {\lt} b-a\text{.}\) Using the density of \(\mathbb {Q}\) in \(\mathbb {R}\text{,}\) we know that there exists \(r\in \mathbb {Q}\) satisfying \(b-\epsilon {\lt} r {\lt} b\text{.}\) Since clearly \(r\in [a,b]\text{,}\) we know \(r\in T\text{.}\)
    🔗
    You could also try to prove that \(b\) is the least upper bound using contradiction: Towards a proof by contradiction, suppose that there exists an upper bound \(u\) of \(T\) such that \(a \leq u {\lt} b\text{.}\) By the density of \(\mathbb {Q}\) in \(\mathbb {R}\text{,}\) there exists a rational \(r\in \mathbb {Q}\) such that \(u {\lt} r {\lt} b\text{.}\) This means \(r\in [a,b]\text{,}\) implying \(r\in T\text{.}\) The fact that \(u\) is an upper bound for \(T\) means \(r \leq u\) giving the contradiction
    🔗
    \begin{equation*} r \leq u {\lt} r. \end{equation*}
    🔗
