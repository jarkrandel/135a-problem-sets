    Problem 1 (Abbott Exercise 2.2.2): Verify, using the definition of convergence of a sequence, that the following sequences converge to the proposed limit.
    🔗
    1. \(\lim \frac{2n+1}{5n+4} = \frac{2}{5}\text{.}\)
      🔗
      🔗
    2. \(\lim \frac{2n^2}{n^3+3} = 0\text{.}\)
      🔗
      🔗
    3. \(\lim \frac{\sin(n^2)}{\sqrt[3]{n}} = 0\text{.}\)
      🔗
      🔗
    Remark: In this problem, Abbott tells you what the limit of the sequence is. How would you figure out what the limit should be yourself?
    🔗
    Solution:
    🔗
    1. Let’s do the preliminary exploration of subtracting 2/5 from the general term of the sequence and trying to find a nice upper bound:
      🔗
      \begin{equation*} \begin{aligned}\bigg|\frac{2n+1}{5n+4} - \frac{2}{5}\bigg| = \bigg|\frac{5(2n+1) - 2(5n+4)}{5(5n+4)}\bigg| = \frac{3}{25n+20} \leq \frac{3}{25n}.\end{aligned} \end{equation*}
      This tells us how we should choose \(N\) in terms of \(\epsilon \) for our proof. Indeed, notice that our upper bound \(3/25n\) satisfies \(3/25n {\lt} \epsilon \) if and only if \(n {\gt} 3/25\epsilon \)
      🔗
      Proof: Let \(\epsilon {\gt} 0\text{.}\) Our goal is to find \(N\in \mathbb {N}\) such that
      🔗
      \begin{equation*} \bigg|\frac{2n+1}{5n+4} - \frac{2}{5}\bigg| {\lt} \epsilon \text{ for every } n \geq N. \end{equation*}
      We claim that any integer \(N {\gt} \frac{3}{25\epsilon }\) is an admissible choice. Indeed, the above calculation shows that for any \(n \geq N\text{,}\)
      🔗
      \begin{equation*} \begin{aligned}\bigg|\frac{2n+1}{5n+4} - \frac{2}{5}\bigg| \leq \frac{3}{25n} {\lt} \frac{3}{25 N} {\lt} \frac{3}{25 (3/25\epsilon )} = \epsilon .\end{aligned} \end{equation*}
      🔗
    2. Let’s do the preliminary exploration of subtracting 0 from the general term of the sequence and trying to find a nice upper bound:
      🔗
      \begin{equation*} \begin{aligned}\bigg|\frac{2n^2}{n^3 + 3}-0\bigg| = \frac{2n^2}{n^3+3}\leq \frac{2n^2}{n^3} = \frac{2}{n}.\end{aligned} \end{equation*}
      Notice that \(\frac{2}{n} {\lt} \epsilon \) if and only if \(n {\gt} \frac{2}{\epsilon }\text{.}\)
      🔗
      Proof: Let \(\epsilon {\gt} 0\text{.}\) We claim that any integer \(N {\gt} 2/\epsilon \) is an admissible choice of \(N\text{.}\) Indeed, if \(n \geq N\text{,}\) then our work above shows
      🔗
      \begin{equation*} \bigg|\frac{2n^2}{n^3 + 3}-0\bigg| \leq \frac{2}{n} \leq \frac{2}{N} {\lt} \frac{2}{2/\epsilon } = \epsilon . \end{equation*}
      🔗
    3. Notice that
      🔗
      \begin{equation*} \bigg|\frac{\sin (n^2)}{n^{1/3}}-0\bigg| \leq \frac{1}{n^{1/3}} \end{equation*}
      and that \(n^{-1/3} {\lt} \epsilon \) if and only if \(n {\gt} \epsilon^{-3}\text{.}\)
      🔗
      Proof: Let \(\epsilon {\gt} 0\text{.}\) We claim that any integer \(N {\gt} \epsilon^{-3}\) is an admissible choice of \(N\text{.}\) Indeed, if \(n \geq N\) then
      🔗
      \begin{equation*} \bigg|\frac{\sin (n^2)}{n^{1/3}}-0\bigg| \leq \frac{1}{n^{1/3}} \leq \frac{1}{N^{1/3}} {\lt} \frac{1}{(\epsilon^{-3})^{1/3}}=\epsilon . \end{equation*}
      🔗
    Problem 2 (Abbott Exercise 2.3.3): (Squeeze Theorem) Show that if \(x_n \le y_n \le z_n\) for all \(n \in \N\text{,}\) and if \(\lim x_n = \lim z_n = \ell\text{,}\) then \(\lim y_n = \ell\) as well.
    🔗
    Solution: It is tempting to try to apply the Order Limit Theorem directly here, but we do not know that \((y_n)\) converges initially. Let’s prove it converges to \(\ell \) directly. We want to control \(|y_n-\ell |\) using the fact that \(|y_n-\ell | {\lt} \epsilon \) is equivalent to \(y_n\in (\ell -\epsilon ,\ell +\epsilon )\) is equivalent to \(\ell -\epsilon {\lt} y_n {\lt} \ell +\epsilon \text{.}\) Our plan is to use the definition of convergence for \(x_n\) and \(z_n\) to show that \(y_n\) converges by bounding \(x_n\) below by \(\ell -\epsilon \) and above by \(\ell +\epsilon \text{.}\)
    🔗
    Fix \(\epsilon {\gt} 0\text{.}\) We need to find \(N\in \mathbb {N}\) such that
    🔗
    \begin{equation*} \ell -\epsilon {\lt} y_n {\lt}\ell + \epsilon \text{ for all } n \geq N. \end{equation*}
    Since \(\lim x_n = \lim z_n = \ell \text{,}\) we can apply the definition of the limit for \(x_n\) and \(z_n\) to get integers \(N_1\) and \(N_2\) such that
    🔗
    \begin{equation*} |x_n - \ell | {\lt} \epsilon \text{ for all } n\geq N_1 \end{equation*}
    and
    🔗
    \begin{equation*} |z_n - \ell | {\lt} \epsilon \text{ for all } n\geq N_2. \end{equation*}
    In particular, these equalities imply \(\ell -\epsilon {\lt} x_n\) and \(z_n {\lt} \ell +\epsilon \) for all \(n\) as above. We claim that any integer \(N \geq \max \{ N_1,N_2\} \) is an admissible choice. Indeed, \(n \geq N\) implies both of the above inequalities hold so that
    🔗
    \begin{equation*} \ell -\epsilon {\lt} x_n \leq y_n \leq z_n {\lt} \ell + \epsilon \end{equation*}
    as desired.

    Problem 3 (Abbott Exercise 2.3.10): Consider the following list of conjectures. Provide a short proof for those that are true and a counterexample for any that are false.
    🔗
    1. If \(\lim(a_n-b_n) = 0\text{,}\) then \(\lim a_n = \lim b_n\text{.}\)
      🔗
      🔗
    2. If \((b_n) \to b\text{,}\) then \(|b_n| \to |b|\text{.}\)
      🔗
      🔗
    3. If \((a_n) \to a\) and \((b_n-a_n) \to 0\text{,}\) then \((b_n) \to a\text{.}\)
      🔗
      🔗
    4. If \((a_n) \to 0\) and \(|b_n-b| \le a_n\) for all \(n \in \N\text{,}\) then \((b_n) \to b\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. The statement of this problem is actually a bit ambiguous. The fact that Abbott writes \(\lim a_n\) and \(\lim b_n\) might lead some to think that you are meant to assume that the limits of both exist from the beginning, in which case the statement is True. My interpretation is that you should not assume the limits exist, in which case the answer is False. Let \(a_n = b_n = n\text{.}\) Neither sequence converges, but their difference is the constant sequence 0.
      🔗
      🔗
    2. True. For proof, recall the triangle inequality! Specifically, recall the fact that \(||a|-|b||\leq |a-b|\text{.}\) Applying this to \(b_n\) and \(b\text{,}\) we get
      🔗
      \begin{equation*} ||b_n|-|b|| \leq |b_n-b| \text{ for every } n\in \mathbb {N}. \end{equation*}
      Let \(\epsilon {\gt} 0\) and apply the definition of \(\lim b_n = b\) to find \(N\in \mathbb {N}\) such that \(n {\gt} N\) implies \(|b_n - b| {\lt} \epsilon \text{.}\) We claim that this same \(N\) is an admissible choice. Indeed, \(n \geq N\) implies
      🔗
      \begin{equation*} ||b_n|-|b||\leq |b_n-b|{\lt} \epsilon \end{equation*}
      as desired.
      🔗
      🔗
    3. True. We can use the Algebraic Limit Theorem twice. Since \((a_n-b_n)\) is convergent, so is \(((-1)(a_n-b_n)) = (b_n-a_n)\text{.}\) The sum of this and \((a_n)\) is a convergent sequence whose limit is the sum of the limits. That is,
      🔗
      \begin{equation*} b_n = (b_n-a_n) + (a_n) \rightarrow 0 + a= a. \end{equation*}
      🔗
    4. True. Let \(\epsilon {\gt} 0\) and use the fact that \(\lim a_n = 0\) to find \(N\in \mathbb {N}\) such that \(n \geq N\) implies \(|a_n| {\lt} \epsilon \text{.}\) We claim this choice of \(N\) is admissible for proving that \(\lim b_n = b\text{.}\) Indeed, for any \(n \geq N\text{,}\)
      🔗
      \begin{equation*} |b_n - b| \leq a_n {\lt} \epsilon . \end{equation*}
      🔗
    Problem 4: Limits of convergent sequences are unique. Let’s prove it in two ways! Let \((a_n)\) be a sequence of real numbers and let \(a\in \mathbb {R}\text{.}\) Suppose that \((a_n)\rightarrow a\text{.}\)
    🔗
    1. Prove that \(a\) is unique by applying Theorem 1.2.6 to prove that if \(b\) is any real number such that \((a_n)\rightarrow b\text{,}\) then \(a = b\text{.}\)
      🔗
      🔗
    2. Prove that \(a\) is unique by contradiction: Suppose that there exists \(b\not=a\) such that \((a_n)\rightarrow b\) in addition to \((a_n)\rightarrow a\) and derive a contradiction.
      🔗
      🔗
    Solution:
    🔗
    1. We need to show that for every \(\epsilon {\gt} 0\text{,}\) \(|a-b| {\lt} \epsilon \text{.}\) The fact that \((a_n)\) comes arbitrarily close to both \(a\) and \(b\) means we can use the triangle inequality to pull \(a\) and \(b\) close together.
      🔗
      Let \(\epsilon {\gt} 0\text{.}\) Since \((a_n)\rightarrow a\) and \((b_n)\rightarrow b\text{,}\) there exist integers \(N_1,N_2\) such that
      🔗
      \begin{equation*} |a_n-a| {\lt} \frac{\epsilon }{2} \text{ for all } n\geq N_1 \end{equation*}
      and
      🔗
      \begin{equation*} |a_n-b|{\lt}\frac{\epsilon }{2} \text{ for all } n\geq N_2. \end{equation*}
      Let \(N = \max \{ N_1,N_2\} \text{.}\) Then \(N\geq N_1\) and \(N \geq N_2\) so that both of the above inequalities hold when \(n = N\text{.}\) Therefore, the triangle inequality implies
      🔗
      \begin{equation*} |a-b| = |(a-a_N) + (a_N-b)| \leq |a-a_N| + |a_N-b| {\lt} \frac{\epsilon }{2} + \frac{\epsilon }{2} = \epsilon . \end{equation*}
      🔗
    2. Assuming that \(a\not=b\text{,}\) we know that \(\epsilon = |a-b| {\gt} 0\text{.}\) This will contradict the fact that \((a_n)\) must eventually get extremely close to both \(a\) and \(b\text{.}\) Indeed, the facts that \((a_n)\rightarrow a\) and \((a_n)\rightarrow b\) mean that there exist \(N_1, N_2\) such that
      🔗
      \begin{equation*} |a_n-a| {\lt} \frac{\epsilon }{2} \text{ for all } n\geq N_1 \end{equation*}
      and
      🔗
      \begin{equation*} |a_n-b|{\lt}\frac{\epsilon }{2} \text{ for all } n\geq N_2 \end{equation*}
      exactly as in the previous part. Choose \(N = \max \{ N_1,N_2\} \text{.}\) Then both of the above inequalities above apply with \(n=N\) so that
      🔗
      \begin{equation*} \epsilon = |a-b| = |(a-a_N) + (a_N-b)| \leq |a-a_N| + |a_N-b| {\lt} \frac{\epsilon }{2} + \frac{\epsilon }{2} = \epsilon . \end{equation*}
      So, we’ve shown that \(\epsilon {\lt} \epsilon \text{,}\) a contradiction!
      🔗
      🔗
