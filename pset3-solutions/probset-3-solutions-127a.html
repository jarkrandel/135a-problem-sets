    Problem 1: Let \((x_n)\) be a sequence such that \(x_1 = 2\) and
    🔗
    \begin{equation*} x_{n+1} = \frac{x_n+3}{2}\text{ for all } n\geq 1 \end{equation*}
    1. Use induction to prove that \(x_n\leq 3\) and \(x_{n+1}\geq x_n\) for all \(n\in \mathbb {N}\text{.}\)
      🔗
      🔗
    2. Now, a lemma: Let \(a\in \mathbb {R}\) and \((a_n)\) be a sequence of real numbers. Prove that if \((a_n)\rightarrow a\text{,}\) then the sequence \((b_n)\) defined by \(b_n = a_{n+1}\) satisfies \((b_n)\rightarrow a\) as well.
      🔗
      🔗
    3. Prove that \((x_n)\) converges and use the result of (b) to find its limit \(x\in \mathbb {R}\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. We know \(x_1 = 2 \leq 3\) and \(x_2 = \frac{x_1+3}{2} = \frac{5}{2} \geq 2\text{,}\) so the base case of the induction holds. Now, suppose that the claim is true for some \(n\in \mathbb {N}\text{.}\) We want to use this to show that it holds for \(n + 1\text{.}\) Notice that
      🔗
      \begin{equation*} x_{n+1} = \frac{x_n+3}{2} \leq \frac{3+3}{2} = 3, \end{equation*}
      using the inductive hypothesis that \(x_n \leq 3\text{.}\) This shows that \(x_{n+1} \leq 3\text{.}\) Also, we have
      🔗
      \begin{equation*} x_{n+2} = \frac{x_{n+1} + 3}{2} \geq \frac{x_n + 3}{2} = x_{n+1} \end{equation*}
      using the inductive hypothesis that \(x_{n+1} \geq x_n\text{.}\)
      🔗
      🔗
    2. To prove \((b_n) \rightarrow a\text{,}\) let \(\epsilon {\gt} 0\) and let \(N\in \mathbb {N}\) such that \(n\geq N\) implies \(|a_n-\ell | {\lt} \epsilon \text{.}\) We claim that this \(N\) suffices. Indeed, for any \(n \geq N\text{,}\) \(|b_n-\ell | = |a_{n+1} - \ell | {\lt} \epsilon \) since \(n+1 {\gt} n \geq N\text{.}\)
      🔗
      🔗
    3. Since \((x_n)\) is bounded above and increasing, the Monotone Convergence Theorem implies that it converges to some real number \(x\text{.}\) Using (b) and the Algebraic Limit Theorem, we can conclude that
      🔗
      \begin{equation*} x = \lim_{n\rightarrow \infty }x_{n+1} = \lim_{n\rightarrow \infty }\frac{x_n + 3}{2}= \frac{\lim_{n\rightarrow \infty }x_n + 3}{2} = \frac{x+3}{2} \end{equation*}
      which says \(x = \frac{x+3}{2}\text{,}\) implying \(x = 3\text{.}\)
      🔗
      🔗
    Problem 2 (Abbott Exercise 2.5.1): Give an example of each of the following, or argue that such a request is impossible.
    🔗
    1. A sequence that has a subsequence that is bounded but contains no subsequence that converges.
      🔗
      🔗
    2. A sequence that does not contain \(0\) or \(1\) as a term but contains subsequences converging to each of these values.
      🔗
      🔗
    3. A sequence that contains subsequences converging to every point in the infinite set \(\{1,1/2,1/3,1/4,1/5,\ldots\}\text{.}\)
      🔗
      🔗
    4. A sequence that contains subsequences converging to every point in the infinite set \(\{1,1/2,1/3,1/4,1/5,\ldots\}\text{,}\) and no subsequences converging to points outside of this set.
      🔗
      🔗
    Solution:
    🔗
    1. Impossible by Bolzano-Weierstrass.
      🔗
      🔗
    2. Use the sequence
      🔗
      \begin{equation*} a_n = \begin{cases} \frac{1}{n+1} & \text{if } n \text{ is even},\\ 1-\frac{1}{n+1} & \text{otherwise}. \end{cases} \end{equation*}
      🔗
    3. Use the sequence
      🔗
      \begin{equation*} 1,\ 1,\ 1/2,\ 1,\ 1/2,\ 1/3,\ 1,\ 1/2,\ 1/3,\ 1/4, \ldots , \end{equation*}
      gotten by repeatedly concatenating the first \(k\) elements of the set for every \(k\text{.}\)
      🔗
      🔗
    4. Impossible. Any such sequence \((a_n)\) must have 0 as a limit of a subsequence. Indeed, for any \(\epsilon {\gt} 0\text{,}\) there are infinitely many \(n\in \mathbb {N}\) such that \(a_n\in (-\epsilon ,\epsilon )\text{.}\) To get a subsequence converging to 0 using this fact, we can inductively choose an increasing sequence \(n_1 {\lt} n_2 {\lt} \ldots \) such that \(a_{n_k}\in (-1/k,1/k)\text{.}\) To see this fact holds, let \(k\in \mathbb {N}\) be such that \(1/k {\lt} \epsilon /2\) and let \((a_{n_m})\) be a subsequence converging to \(1/k\text{.}\) There are infinitely many \(m\in \mathbb {N}\) such that \(|a_{n_m}-\frac{1}{k}| {\lt} \frac{\epsilon }{2}\text{,}\) meaning \(a_{n_m} \leq 1/k + \epsilon /2 \leq \epsilon \) and \(a_{n_m} \geq 1/k-\epsilon /2 \geq -\epsilon \text{.}\) So, there are infinitely many \(m\) such that \(a_{n_m}\in (-\epsilon ,\epsilon )\) as desired.
      🔗
      🔗
    Problem 3 (Abbott Exercise 2.5.5): Assume \((a_n)\) is a bounded sequence with the property that every convergent subsequence of \((a_n)\) converges to the same limit \(a \in \R\text{.}\) Show that \((a_n)\) must converge to \(a\text{.}\)
    🔗
    Solution: One strategy is to use proof by contradiction: If the sequence does not converge to \(a\text{,}\) then we can find infinitely many terms outside of a neighborhood of \(a\text{.}\) These form a subsequence of \(a_n\) that, by Bolzano-Weierstrass, converge to some number not equal to \(a\text{.}\)
    🔗
    Proof: Towards a contradiction, assume that \(a_n\) does not converge to \(a\text{.}\) Being careful with the quantifiers, this means that there exists \(\epsilon {\gt} 0\) such that for all \(N\in \mathbb {N}\text{,}\) there exists some \(n \geq N\) such that \(|a_n-a| \geq \epsilon \text{.}\) Applying this condition once (say, with \(N=1\)), we receive an integer \(n_1\) such that \(|a_{n_1}-a|\geq \epsilon \text{.}\) Applying this condition with \(N = n_1+1\text{,}\) we receive an integer \(n_2 {\gt} n_1\) such that \(|a_{n_2}-a| \geq \epsilon \text{.}\) Continuing in this fashion, we receive a subsequence \((a_{n_k})\) such that
    🔗
    \begin{equation*} |a_{n_k}-a| \geq \epsilon \text{ for all } k\in \mathbb {N} \end{equation*}
    Since \((a_n)\) is bounded, \((a_{n_k})\) is bounded, and Bolzano-Weierstrass implies that \((a_{n_k})\) itself has a subsequence \((c_m)\) that converges to \(a\text{.}\) This contradicts the fact that \(|c_m-a|\geq \epsilon \) for all \(m\in \mathbb {N}\text{,}\) completing the proof.
    🔗
    Problem 4 (Abbott Exercise 2.6.3): If \((x_n)\) and \((y_n)\) are Cauchy sequences, then one easy way to prove that \((x_n+y_n)\) is Cauchy is to use the Cauchy Criterion. By Theorem 2.6.4, \((x_n)\) and \((y_n)\) must be convergent, and the Algebraic Limit Theorem then implies \((x_n+y_n)\) is convergent and hence Cauchy.
    🔗
    1. Give a direct argument that \((x_n+y_n)\) is a Cauchy sequence that does not use the Cauchy Criterion or the Algebraic Limit Theorem.
      🔗
      🔗
    2. Do the same for the product \((x_n y_n)\text{.}\)
      🔗
      🔗
    Solution:
    🔗
    1. Let \(\epsilon {\gt} 0\text{.}\) We need to produce an \(N\in \mathbb {N}\) such that \(n,m \geq N\) imply \(|x_n+y_n-x_m-y_m| {\lt} \epsilon \text{.}\) Using the fact that \((x_n)\) and \((y_n)\) are Cauchy, we can find \(N_1,N_2\in \mathbb {N}\) such that
      🔗
      \begin{equation*} |x_n-x_m| {\lt} \frac{\epsilon }{2} \text{ whenever } n,m\geq N_1 \end{equation*}
      and
      🔗
      \begin{equation*} |y_n-y_m| {\lt} \frac{\epsilon }{2} \text{ whenever } n,m \geq N_2. \end{equation*}
      We claim that \(N = \max \{ N_1,N_2\} \) is an admissible choice of \(N\text{.}\) Indeed, for any \(n,m\geq N\text{,}\) the triangle inequality gives
      🔗
      \begin{equation*} |x_n+y_n-x_m-y_m| = |(x_n-x_m)+(y_n-y_m)| \leq |x_n-x_m| + |y_n-y_m| {\lt} \frac{\epsilon }{2}+\frac{\epsilon }{2} = \epsilon . \end{equation*}
      🔗
    2. Let’s recall some algebra used in bounding the difference of products. Recall
      🔗
      \begin{equation*} \begin{aligned}|x_ny_n - x_my_m| & = |x_ny_n-x_ny_m +x_ny_m-x_my_m|\\ & = |x_n(y_n-y_m) + y_m(x_n-x_m)|\\ & \leq |x_n||y_n-y_m| + |y_m||x_n-x_m|\end{aligned} \end{equation*}
      where we used the triangle inequality in the final line. Since \(x_n\) and \(y_n\) are Cauchy, they are bounded, so we can replace the \(|x_n|\) and \(|y_m|\) with uniform bounds. The differences can clearly be made small as well, leading to the result. Let’s implement this plan.
      🔗
      Proof: Let \(\epsilon {\gt} 0\text{.}\) We need to produce \(N\in \mathbb {N}\) such that \(n,m \geq N\) imply \(|x_ny_n-x_my_m| {\lt} \epsilon \text{.}\) Since \((x_n)\) and \((y_n)\) are Cauchy, they are bounded. This means there exist constants \(M_1,M_2 {\gt} 0\) such that
      🔗
      \begin{equation*} |x_n| \leq M_1 \text{ for all } n\in \mathbb {N} \end{equation*}
      and
      🔗
      \begin{equation*} |y_n| \leq M_2 \text{ for all } n\in \mathbb {N}. \end{equation*}
      Using the definition of Cauchy, we can also find \(N_1,N_2\in \mathbb {N}\) such that
      🔗
      \begin{equation*} |x_n-x_m| {\lt} \frac{\epsilon }{2M_2} \text{ for all } n,m\geq N_1 \end{equation*}
      and
      🔗
      \begin{equation*} |y_n-y_m| {\lt} \frac{\epsilon }{2M_1} \text{ for all } n,m\geq N_2. \end{equation*}
      We choose \(N = \max \{ N_1,N_2\} \text{.}\) To verify that this \(N\) works, let \(n,m\geq N\) and observe that
      🔗
      \begin{equation*} \begin{aligned}|x_ny_n - x_my_m| & \leq |x_n||y_n-y_m| + |y_m||x_n-x_m|\\ & {\lt} M_1 \frac{\epsilon }{2M_1} + M_2\frac{\epsilon }{2M_2}\\ & = \frac{\epsilon }{2} + \frac{\epsilon }{2} = \epsilon .\end{aligned} \end{equation*}
      🔗
